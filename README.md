# Tech4Good

## Dev environement ##
To launch local version of Wordpress go to compose directory and launch command  : 

```shell script
docker-compose up -d
```
#### Urls : #### 
- Wordpress : http://localhost:8000
- Wordpress Admin : http://localhost:8000/wp-admin
